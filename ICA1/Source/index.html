  • Step2 is to choose plan
  • Step3 is to Tailor experience

Git Hub Repository Creation

Below are the steps performed for Git Hub Repository creation

  1. Sign in in the Git Hub : https://github.com/
  2. Create new repository by clicking on the + icon
  3. Open Git Hub Desktop and clone the repository
  4. Open folder and create two folders Source and Documentation
  5. Create source files in Source folder and screenshots in the documentation
  6. Commit the files
  7. Sync to the remote repository
  8. Check for the result in the GitHub website
